# Project Overview

Space Mission Explorer is a responsive web application developed using HTML5, CSS3, and JavaScript. The application allows users to explore upcoming and recent space launches, track live countdowns, view rocket information, and stay updated with the latest space news through real-time API integration.

# Application Sections

### Home Section

Displays the introduction and navigation area of the application.

### Live Clocks Section

Shows both UTC Time and Local Time in real time.

### Statistics Section

Displays total launches, agencies, locations, and rockets.

### Space Missions Section

Shows upcoming and recent launches with mission status and countdown timers.

### Rocket Gallery Section

Displays rocket information fetched from the SpaceX API.

### Latest Space News Section

Shows the latest space-related news articles.

### Mission Details Modal

Provides detailed information about selected missions.

# API Overview

### Launch Library 2 API

Provides real-time information about upcoming and previous space launches. It supplies mission details, launch status, launch providers, and launch locations used throughout the application.

**Endpoint Used:**

https://ll.thespacedevs.com/2.2.0/launch/

---

### SpaceX API

The data is used to build the Rocket Gallery section.

**Endpoint Used:**

https://ll.thespacedevs.com/2.2.0/config/launcher/?limit=50

---

### Spaceflight News API

Provides the latest space and aerospace news articles from various trusted sources. It is used to display recent news updates within the application.

**Endpoint Used:**

https://api.spaceflightnewsapi.net/v4/articles/?limit=6
